## [демо](http://95.217.10.202:3000/)

## Как развернуть

### backend
- Создать файл .env согласно образцу .env.example с указанием ключа для reCaptcha
- npm i
- npm run start

### frontend
- Создать файл .env согласно образцу .env.example с указанием ключа для reCaptcha и url бэкенда
- npm i
- npm run build
- npm run start
